Tutorial

Step 1: Changing the Menu

This is pretty straight forward and very simple. After creating your navbar you create a ul tag and then create a list. (li tags) You can add how many you want.
After you want to create a hyperlink tag (< a >) which allows the user to click on your menu and go to the links destination. Lastly you type in whatever you want between both A tags and you're set!

Step 2: Changing the Header


There are other ways to change the header picture, but for me particularly I used the CSS file to hold my image.
Therefore if you're doing it this way to change the picture you must go into the create.css file and scroll to the header class. In my case its under headmast.head and then inside the brackets of "background-image: url(this is where you image file goes)"

Step 3: Creating Who Am I? Page


After setting up my section and div tags in order to make the card you must use class="card". Also you can edit how big you want it with style.
Next you want to upload your image, but this time you can just use the img tag inside the index instead of going into the css file. Lastly you just input your information you want inside the card and you're done!

Step 4: Creating Skills Page


When creating my skills page I again made my sections, div tags. (look below for reference) When creating the images and descriptions I just used an img tag again, but this time with the rounded class in order to make my images circular.
Next I just edited the height and width of the images and lastly used the p tag to insert the description. Then it's just rinse and repeat for how many skills I had.

Step 5: Creating My projects Page


For this page it's almost identical of what I did with my skills page. The only difference is that I used the hyperlink tag so that when you click on the image it will take you to a close up of the picture.

Step 6: Creating Contact Page


For creating this contact page all I did was use a p tag to have a little description and under it make a hyperlink tag so that when the user clicks on my email they will be able to email me using the mailto function.
